## Descripción General del Proyecto

Una pequeña tienda desea contar con un programa básico en C++ que le permita gestionar sus productos. El dueño de la tienda necesita:
Registrar productos.

Mostrar la lista de productos.
Ordenarlos por precio, nombre o cantidad utilizando métodos de ordenamiento básicos.
Actualizar información de un producto seleccionado.
Buscar un producto por nombre.
Generar un pequeño reporte final.

El sistema deberá construirse aplicando los temas del curso y ajustándose a las funcionalidades obligatorias descritas más adelante.

---

## Funcionalidades del Sistema

### 1. Registrar productos
Permite ingresar:
- Nombre  
- Precio  
- Cantidad  

Incluye validaciones para evitar datos incorrectos (precio y cantidad mayores a 0).

---

### 2. Mostrar productos
Despliega en pantalla una tabla con:
- Nombre  
- Precio  
- Cantidad  

Solo se muestran productos válidos.

---

### 3. Ordenar productos
El sistema puede ordenar los productos **por precio (ascendente)** utilizando tres algoritmos:

- Método Burbuja  
- Método Selección  
- Método Inserción  

Después del ordenamiento, se imprime la lista ordenada.

---

### 4. Buscar producto por nombre
Permite encontrar un producto ingresando su nombre exactamente como fue registrado.

---

### 5. Actualizar producto
El usuario puede seleccionar un producto por índice y modificar:
- Precio  
- Cantidad  

Ambos valores deben ser mayores a cero.

---

### 6. Generar reporte final
El programa calcula:
- Total de productos registrados  
- Suma total de los precios en inventario  
- Producto con mayor cantidad
